# Lit checkbox-switcher

## Installation

```sh
npm i @ui-wc/checkbox
```

## Usage

```ts
import {WCCheckbox} from '@ui-wc/checkbox'

WCCheckbox.define()

// ...
```

## Examples

```html
<lit-checkbox type = "checkbox"></lit-checkbox>

<!-- Switcher is defaul type -->
<lit-checkbox></lit-checkbox>
```
