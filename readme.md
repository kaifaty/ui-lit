# UI lit

Web-components UI library based on Lit 2.0

## [Storybook](https://kaifaty.github.io/ui-lit/storybook-static)

## Features

- Small size: **~30kb** gzip
- Based on web-components
- Form-associated elements

## Getting started

```npm i ui-lit```

```yarn add ui-lit```

## Motoivation

To build fast, simply to use UI lib with minimum requirements.
